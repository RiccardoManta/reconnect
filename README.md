# Chassis ReConnect

A local network server connection management application built with Next.js and TypeScript.

## Features

- View and manage servers on the local network
- Add new servers with custom names, categories, and descriptions
- Connect to online servers
- Monitor server status (online/offline)

## Getting Started

### Prerequisites

- Node.js 18.0.0 or later
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Start the development server:

```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Technologies Used

- Next.js
- TypeScript
- Tailwind CSS
- React Icons
- Lucide React
